# Errors

## `AI_CHAT_MEMORY_DEBUG_INVALID_INPUT`

- Invalid or unexpected body/query payload.

## `AUTH_INVALID_SESSION`

- Missing or invalid authenticated session.

## `USER_PROFILE_NOT_FOUND`

- Authenticated user does not map to a `UserProfile`.

## `AI_CHAT_MEMORY_DEBUG_INTERNAL_ERROR`

- Unexpected internal failure.
